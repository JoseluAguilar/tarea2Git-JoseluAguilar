MINEVANCIC

Rules of the game

Rover is our endearing robot that has a mission in this game

Rover has to go to the exit box to be able to pass the level. To do this you have to get to the exit without stepping on any bomb and before time runs out

The playing field will increase in size as the levels pass and the number of mines will also increase

If the mine is a:

  • 3 or more boxes (Rover will be green)
  • 2 boxes (Rover will be yellow)
  • 1 box (Rover will be red)

Rover can't retrace his steps so think well before you move him

You can use the game menu to move Rover but you can also use the W S A D keys

Enjoy the game and thanks for playing the game of Rover
